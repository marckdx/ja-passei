# Já-Passei
<p>Copyleft 2014</p> 

<p>Sistema para Android que calcula o quanto o aluno precisa tirar em média nas provas para que obtenha a média mínima necessária.</p>

## Notas da versão
<p>Versão atual: 2.4 (01/12/2014)</p>
